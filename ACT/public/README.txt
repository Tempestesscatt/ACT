COM PUBLICAR-HO
1. Entra a GitHub > ACT > ACT > public.
2. Elimina o substitueix l'index.html antic.
3. Puja TOTS els fitxers i la carpeta assets mantenint aquesta estructura.
4. Fes Commit changes a la branca main.
5. Espera el desplegament de Vercel i recarrega amb Cmd + Shift + R.

IMPORTANT
- No pugis només index.html: les altres pàgines i la carpeta assets són necessàries.
- Les fotografies meteorològiques són externes i reals, servides des de Wikimedia Commons.
- El formulari és provisional i no envia dades.
- Revisa els textos legals i les dades de l'associació abans del llançament oficial.
